## 📋 About This Project

A fully functional real estate property listing website built with PHP, MySQL, HTML, CSS, and JavaScript.

### Features:
- 🔍 Property search and filtering
- 📋 Detailed property listings with images
- 👤 User registration and login system
- 📱 Responsive design (works on all devices)
- 📧 Contact forms for property inquiries
- 🏷️ Property categories and filters
- 🔐 Admin panel for managing listings

---

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache (AwardSpace)
